🚀 3-STEP DEPLOYMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Create BBB Server (10 minutes)
───────────────────────────────────────

  1. Sign up: https://www.hetzner.com/cloud
  
  2. Create server:
     • Type: CPX21
     • RAM: 4GB
     • Price: €5.83/month (~$6)
     • Location: Choose closest to students
     • Image: Ubuntu 20.04
  
  3. Point domain:
     • Add DNS A record: bbb.rv2class.com → SERVER_IP
     • Wait 5-10 minutes

  4. SSH into server:
     ssh root@YOUR_SERVER_IP

  5. Install BBB (one command, takes 15 min):
     
     wget -qO- https://ubuntu.bigbluebutton.org/bbb-install-2.7.sh | \
       bash -s -- -w -v focal-270 \
       -s bbb.rv2class.com \
       -e your@email.com

  6. Get credentials:
     bbb-conf --secret
     
     Save the URL and Secret that appear!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: Deploy to Vercel (2 minutes)
─────────────────────────────────────

  cd /home/roman/Documents/rv2class

  export BBB_URL="https://bbb.rv2class.com/bigbluebutton/"
  export BBB_SECRET="your_secret_from_step_1"

  ./deploy-vercel-bbb.sh

  That's it! Script handles everything! 🎉

🆘 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: DNS not resolving
  Fix: Wait 10-15 minutes after adding DNS record

  Problem: BBB install fails
  Fix: Make sure you used Ubuntu 20.04, not 22.04

  Problem: Can't access BBB
  Fix: Check firewall allows ports 80, 443, 16384-32768/udp

  Problem: Video doesn't work
  Fix: Run on server: bbb-conf --check
